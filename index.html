js数据类型及私有属性

  • Array数组类型 push()：向数组的末尾添加新的元素 concat()：合并两个数组或个数组 pop()：删除数组的最后一个元素 reverse()：反转数组 sort()：字母顺序排列数组 sort(function(a,b){return a-b})：数字逆序排列数组 sort(function(a,b){return b-a})：数字逆序排列数组 slice()：选择数组中的第到y个元素 splice()：在数组中指定位置添加新的元素 unshift()：在数组开头添加新元素 shift()：删除数组的第一个元素
  • Number数字类型 toFixed()：把数字转换为字符串，结果的小数点后有指定位数的数字 toPrecision()：把数字格式化为指定的长度
  • String字符串类型 charAt()：返回指定位置的字符 concat()：连接两个字符串或多个字符串 match()：查找找到一个或多个正则表达式的匹配 replace()：在字符串中查找匹配的子串，并替换与正则表达式匹配的子串 search()：查找与正则表达式相匹配的值 slice()：提取字符串的片断，并在新的字符串中返回被提取的部分 split()：把字符串分割为字符串数组 toLowerCase()：把字符串转换为小写 toUpperCase()：把字符串转换为大写 trim()：去除字符串两边的空白 toString()：返回一个字符串
  • Date日期与时间类型 getFullYear()：从Date对象以四位数字返回年份 getDay()：从Date对象返回一周中的某一天(0 ~ 6) getDate()：从Date对象返回一个月中的某一天(1 ~ 31) getTime()：返回1970年1月1日至今的毫秒数 toUTCString()：把Date对象的日期部分转换为字符串
  • Math算术类型 round()：对一个数字四舍五入 random()：返回0-1之间的一个随机数 random()*x：返回0-x之间的一个随机数 max()：返回给定数中最大的数 min()：返回给定数中最小的数
  • RegExp正则表达式类型 /搜索内容/i：不区分大小写匹配 /搜索内容/g：全文搜索，找到所有匹配内容 test()：搜索字符串指定的值，根据结果并返回真或假 exec()：检索字符串中的指定值，返回值是被找到的值，如果没有发现匹配，则返回null

获取DOM元素的方法

  • 通过 id 查找 HTML 元素: document.getElementById()
  • 通过标签名查找 HTML 元素: document.getElementsByTagName()
  • 通过类名找到 HTML 元素: document.getElementsByClassName()
